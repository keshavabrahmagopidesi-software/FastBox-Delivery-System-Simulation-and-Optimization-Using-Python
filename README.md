# FastBox-Delivery-System-Simulation-and-Optimization-Using-Python
Python-based delivery simulation system that assigns packages to nearest agents using Euclidean distance, calculates efficiency, and generates JSON/CSV reports for performance analysis.
🚚 Delivery Optimization & Agent Simulation

A Python project that simulates package delivery by assigning the nearest agent to each warehouse using distance-based optimization.

🚀 Features

Select JSON input file via file dialog

Assign packages to nearest delivery agent

Distance calculation using Euclidean formula

Simulate delivery delays

Generate:

📄 JSON performance report

📊 CSV for best agent

ASCII route visualization in console

⚙️ How It Works

Reads input JSON containing:

Agents

Warehouses

Packages

Finds nearest agent for each warehouse

Calculates:

Total distance traveled

Number of deliveries

Efficiency score

Identifies best performing agent
